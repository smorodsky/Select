h2. Select

Extension of Adobe Illustrator for select objects by characteristic.

Using the menu commands File/Scripts/Select for select objects on current document.

Hold down Shift to add to the selection.
Hold down Alt to subtract another selection.
Hold down Alt+Shift to select only objects intersected by other selections.


h3. Installation

Download the file _select.zxp_ and double-click it. Follow the instructions of Adobe Extension Manager.

h3. Uninstall

Run Adobe Extension Manager. Find the item "Select" in the list and click "Remove" button.
